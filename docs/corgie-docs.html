What is Corgie?

Corgie is a CSS/Javascript framework similar to Bootstraps or Foundation. Its built on modern CSS using Flexbox, Filters, and CSS Variables. There is no grid system as I believe grids are best handled by the new native CSS grid. The Javascript is ES6 and all vanilla. No JQuery here. Corgie focuses on clean maintainable code. Instead of thousands of divs it attempts to keep HTML as minimal as possible and instead leverage modern technology in order to create the best development experience.
